# OPCRIT+ Web

Aplicacion web estatica basada en la Tabla 1 del suplemento de Rucker 2011 para captura estructurada OPCRIT+.

Sitio:

https://jaflomd.github.io/opcrit-web/

## Alcance

- Captura completa de las secciones e items hoja de la Tabla 1.
- Salidas diagnosticas alineadas con las categorias de la Tabla 2 mediante reglas transparentes aproximadas.
- Los datos se guardan localmente en el navegador mediante `localStorage`.

Las reglas diagnosticas no equivalen al algoritmo propietario OPCRIT+ original.
